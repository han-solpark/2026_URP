추천 학습 로드맵

Phase 1: 기초 다지기

예상 기간: 4-6주

  • Python 프로그래밍 기초 및 문법
  • 자료구조와 알고리즘 입문
  • 수학 기초 (선형대수, 확률/통계)

Phase 2: 머신러닝 기초

예상 기간: 6-8주

  • 머신러닝 개념과 원리 이해
  • NumPy, Pandas 데이터 처리
  • Scikit-learn을 활용한 기본 모델 학습
  • 회귀, 분류 알고리즘 실습
  • 간단한 ML 프로젝트 진행

Phase 3: 딥러닝 입문

예상 기간: 8-10주

  • 신경망의 기본 원리 이해
  • TensorFlow 또는 PyTorch 기초
  • CNN (이미지 처리)
  • RNN/LSTM (시계열 데이터)
  • 딥러닝 프로젝트 (이미지 분류 등)

Phase 4: 심화 및 응용

예상 기간: 지속적

  • 최신 논문 읽기 및 구현
  • Transformer, Attention 메커니즘
  • 생성 모델 (GAN, VAE 등)
  • 강화학습 기초
  • 종합 프로젝트 및 포트폴리오 구축
